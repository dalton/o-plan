Experimental O-Plan demonstrations

Welcome to the O-Plan experimental Web demo page. From here you can access O-Plan Web demos that aren't ready for general release. You can also see the demos that have been released. This page has them all.

Demonstrations

There are two basic kinds: task-index demonstrations in which you're presented with a list of domains and tasks from which you can select by following a link, and specialized demonstrations, each with its own HTML form.
  • Task-index demonstrations
    • Standard TF demonstrations
    • Standard TF demonstrations embedded in a document
    • Resource demonstrations
    • Getting to work
  • Specialized demonstrations
    • HTTP server matrix demos
      • Pacifica COA matrix server (with replan and two-user model)
      • DERA version of the above
      • Pacifica COA matrix server (original version) (no longer available)
    • Start an HTTP server (if you know what you're doing)
    • CGI matrix demos
      • Pacifica COA matrix (official version)
      • Pacifica COA matrix (the original -- starts with a situation-definition form)
    • Island rescue
    • Island rescue with the user able to turn off the feasibility filter and to use the "use" CM.
    • Volume groups
    • Simple TPN
    • Run O-Plan interactively (doesn't work; never will)
