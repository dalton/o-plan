What's new

October 1997

  • A new COA evaluation matrix demonstration.

May - July 1997

  • A new demonstration that allows you to construct a COA evaluation matrix for disaster relief tasks on Pacifica.
  • A new map of Pacifica.
  • A revised form for the Island Rescue demonstration that should be easier to use.
  • A new demonstration that has O-Plan produce a shell script for removing a volume group (an entity that exists in some versions of Unix).

20 March 1996

  • Fixed a number of recent problems.

19 June 1995

  • A hypertext version of the O-Plan Demonstration Guide is now available. It contains links that run O-Plan, embedded in descriptions of the corresponding problem domains.

21 November 1994

  • The demonstrations now run on a faster and less heavily loaded machine.
  • The "task-index" demonstrations now let you look at the task and schema definitions before you request a plan.
  • Some of the pages under construction are further along. O-Plan on the Internet now describes what it will contain when finished.
  • The O-Plan Demonstration Guide has been linked-in. It's most useful with the Standard TF demonstrations . [Note: you should now use the hypertext version of the Guide instead. See the entry for 19 June 1995.]

18 November 1994

  • Web-demo page announced to O-Plan users.
