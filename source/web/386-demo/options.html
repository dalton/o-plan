O-Plan demo Options
Welcome to the Spottisvax O-Plan Web demo Options page. From here you can access some of the O-Plan Web demos that run on Spottisvax. You can also reach a description of how O-Plan can be used on the Internet.

If you were getting the demos from the AIAI Web-server, you would see a different page. Many of the links reachable from that page will not work here. You might also see the AIAI experimental page.

Demonstrations

There are two basic kinds: task-index demonstrations in which you're presented with a list of domains and tasks from which you can select by following a link, and specialized demonstrations, each with its own HTML form.

    • HTTP server demos

      • Start an HTTP server
    • CGI-based matrix demos
      • Go places and do things in Pacifica
      • The original Pacifica COA matrix
    • Utilities
      • TF syntax-checker
    • Task-defining forms
      • Small Unit Operations (SUO)
      • Island rescue
      • Simple TPN
      • London underground
